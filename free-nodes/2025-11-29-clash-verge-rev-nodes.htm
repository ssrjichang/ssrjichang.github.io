一、什么是 Quantumult X？

Quantumult X 是一款专注于网络规则分流与代理管理的高级工具。它的核心用途是：科学上网、网络加速、规则控制、流量监控、代理链路管理等。由于其强大的灵活性和稳定性，在 iOS 用户圈中广受推崇，尤其适合中高级用户或网络安全爱好者。

它不同于一般的 VPN 客户端，它更像一个小型的可编程网络路由器，你可以在里面设定谁走代理、谁走直连，甚至为不同的流量制定多级规则。这种“规则优先”的架构，让它在应对复杂网络环境时游刃有余。

二、为什么选择 Quantumult X？

在 iOS 平台上，类似的工具还有 Shadowrocket、Loon、Stash 等，但 Quantumult X 能脱颖而出，靠的正是其细腻的功能配置能力和稳定性极高的运行机制。

2.1 核心优势

  • 策略组支持：可以设置多个策略组（如自动选择、节点列表、负载均衡等）。

  • 脚本支持：通过 JavaScript 执行高级功能（如重写请求、广告拦截等）。

  • 流量分析：按时间、应用、节点等多维度进行网络使用统计。

  • 规则灵活：内建强大的“规则分流系统”，用户可轻松自定义规则优先级。

  • 轻量美观：UI 简洁，几乎无卡顿，后台运行稳定。

五、规则与策略组：玩转分流的关键

Quantumult X 的灵魂，在于分流规则与策略组配置。这是它与普通 VPN 最大的差异。

5.1 什么是策略组？

策略组是一个“逻辑集合”，用来指示某类请求要使用哪种连接方式。常见的策略组包括：

  • 直连（DIRECT）：不通过代理，直接连接；

  • 全局代理（PROXY）：所有请求走代理节点；

  • 节点列表：列出多个节点供用户选择；

  • 自动选择（AUTO）：根据延迟或负载自动切换节点；

  • 拒绝（REJECT）：拦截特定流量，常用于广告屏蔽。

5.2 自定义规则分流

你可以根据域名、IP、应用名等创建详细规则。例如：

plaintext
DOMAIN-SUFFIX,google.com,PROXY
DOMAIN-SUFFIX,baidu.com,DIRECT
PROCESS-NAME,YouTube,PROXY

这些规则将自动应用在访问过程中，实现精准控制。

✅ 小贴士：建议使用“规则分组 + GEOIP + DOMAIN”组合，可以最大程度地控制代理策略，防止误杀或漏判。

七、使用中常见问题与解决方案

7.1 Quantumult X 无法连接？

  • 检查 VPN 权限是否授权；

  • 确保节点可用（ping 不通的换个节点）；

  • 检查配置文件是否正确生效。

7.2 VPN 状态频繁断开？

  • 避免开启低功耗模式；

  • 设置中允许后台活动；

  • 尽量使用稳定节点，不建议使用公共免费节点。

7.3 如何重置 Quantumult X？

在设置页中选择“恢复出厂设置”，即可清除所有配置，回归初始状态。

7.4 能否在一个设备上安装多个版本的 Quantumult X？

官方并未推出多版本，因此只能安装一个。若需切换配置，建议保留多个 .qxconfig 文件，根据场景导入即可。

FAQ

Clash 的自定义规则组能实现哪些功能？
Clash 的自定义规则组允许用户将多个节点和策略组合，通过优先级和条件分流流量。这样可以根据网站类别或流量类型自动选择最优节点，提高网络效率和使用体验。
Trojan 如何实现伪装？
Trojan 使用 TLS 协议模拟正常的 HTTPS 流量，防火墙难以区分其与普通网站访问的区别。通过伪装域名、SNI 和证书匹配机制，它能在保障安全的同时隐藏真实代理流量。
Clash 支持多配置文件加载吗？
Clash 支持多配置切换，用户可在图形界面中导入多个 YAML 文件，并通过选择不同配置快速切换使用场景。例如家用模式、移动模式或公司网络模式。
Netch 如何结合节点测速实现自动分流？
Netch 可以结合节点延迟检测和策略组，将不同类型的流量分配到低延迟节点。提高网页、视频和游戏访问速度及稳定性，优化整体网络体验。
Clash 如何防止规则文件冲突导致分流异常？
合理组织规则顺序，将更具体的规则放在前面，通配规则放在末尾；使用 rule-providers 分离不同来源的规则，避免重复条目，并在变更后使用日志或 debug 模式检验分流是否按预期生效。
Clash 的节点健康检测如何工作？
Clash 会定期对每个节点进行延迟和丢包率测试，并根据结果标记节点健康状态。结合策略组，可以自动切换到最优节点，避免访问高延迟或失效节点，提高整体网络稳定性和效率。
Mellow 的 TUN 模式与 SOCKS5 代理有什么区别？
TUN 模式拦截系统 IP 层流量，实现全局代理；SOCKS5 代理通常只影响支持代理的应用。TUN 更适合全局加速和游戏、视频等应用，保证所有流量通过节点，提高网络稳定性。
SSR 的协议参数和混淆参数如何配置最安全？
选择 AEAD 或协议插件可提升安全性，混淆方式如 tls1.2_ticket_auth 可以模拟正常 HTTPS 流量。合理搭配协议和混淆参数，能够有效抵御 DPI 检测和流量封锁。
Netch 如何实现 UDP 游戏加速？
Netch 支持 UDP 转发，可专门为游戏流量建立高速通道。结合策略组和节点延迟检测，可有效减少丢包和延迟，提高游戏体验的流畅性和稳定性。
VLESS + Reality 模式如何隐藏流量特征？
Reality 模式通过伪装握手和真实网站特征，使流量难以被 DPI 检测。结合策略组使用，可有效隐藏节点通信特征，同时保持安全和高速传输，适合高封锁环境。
